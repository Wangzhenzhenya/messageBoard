### 后端
  - couchDB

### 启动 
   ##### `npm start`
   > 浏览器打开 http://localhost:3000 浏览该项目

### 功能
  - 留言
  - 删除留言
  - 留言回复
  - 删除留言回复
  - 创建数据库
       > 当项目启动但无法自动创建数据库时，使用 cmd 创建数据库
       > `npm run createDB`  
  - 删除数据库   
       >`npm run destoryDB`
  - gravatar
       > 可以获取开发者的头像,但没有在gravatar注册邮箱的头像将会显示默认图片
